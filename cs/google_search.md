# Google Advanced Search

**Normal search**

`belgrade serbia`

`@realdonaldtrump`

**Exclude words from your search**

`jaguar speed -car`

`"tallest building"`

**Search for wildcards or unknown words**

`largest * in the world`

```
serbia site:www.rt.com

```

**Search for related sites**

`related:apple.com`

**Search by file type**
